Some of the Key Features of Python 

    1. Dynamically Typed Language
        Python is a dynamically typed language, which means we do not need to declare the data type of a variable.
        In languages like C, we must specify the type:
            int i = 10;
        In Python, we can directly assign values:
            i = 10
            i = "hello"
            i = 15.5
        The type is automatically handled by Python at runtime.

    2. User-Friendly Language

        Python is very easy to write and understand.

        Example in C:
            #include <stdio.h>

            int main() {
                printf("Hello World");
                return 0;
            }   

        Example in Python:
            print("Hello World")

        Python needs less code to do the same task.

    3. Readability
        Python code is easy to read because:
            Simple syntax
            Uses indentation instead of brackets
            Looks close to the English language
        This makes Python code easy to understand and maintain.

    4. Developer Productivity

        Python helps developers work faster.
            Best language for scripting
            Strong community support
            Large number of libraries
            Supports Rapid Application Development (RAD)
